# Tic-Tac-Toe Game

This is a simple implementation of the classic **Tic-Tac-Toe** game for two players. Players alternate turns to place `O` and `X` on a 3Ã—3 grid. The first player to get three of their marks in a row (vertically, horizontally, or diagonally) wins. If all spaces are filled without a winner, it's a **draw**.

## ğŸ•¹ï¸ How to Play

1. **Player 1** starts with `O`.
2. Players take turns clicking on empty squares.
3. After each move:
   - The board updates with the symbol.
   - The game checks for a win or a draw.
4. If a player wins or the game ends in a draw, a message is shown.
5. Use the **Reset Game** button to start over.
